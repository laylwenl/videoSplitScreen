# video-split-screen - Example

这是一个最小示例项目，用来演示如何安装并使用 `video-split-screen` 包（本地示例使用 `file:` 引用本地包）。

运行：

```bash
cd video-split-screen-example
npm install
npm run dev
```

然后打开 `http://localhost:5173` 查看示例页面。

说明：
- 项目通过 `video-split-screen` 本地依赖引用 `file:../video-split-screen-package`，若你已将包发布到 npm，请在 `package.json` 中替换为正式包名。
